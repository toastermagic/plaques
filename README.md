# plaques

This is a project using angular 2 and d3 to visualise infomation from http://openplaques.org

To see it:

`git clone https://github.com/toastermagic/plaques.git`

`cd plaques`

`npm i`

`typings install`

If you use VSCode:

`code .`

hit F5

otherwise:

`gulp build`

`node dist/server/app.js`

serves on `http://localhost:5000`
